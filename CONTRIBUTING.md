# Contributing to Moekyawaung-portfolios-V3

Thank you for your interest in contributing! 🙏  
This is a personal portfolio, but suggestions, bug reports, and improvements are welcome.

---

## 🐛 Reporting Bugs

1. Check [existing issues](../../issues) to avoid duplicates
2. Use the **Bug Report** template
3. Include browser, OS, and device information
4. Attach screenshots if relevant

## ✨ Suggesting Features

1. Open a [Feature Request](../../issues/new?template=feature_request.md)
2. Describe the problem it solves
3. Include mockups or references if possible

## 🔧 Making Changes

1. **Fork** the repository
2. Create a feature branch: `git checkout -b feature/your-idea`
3. Make your changes in `index.html`
4. Test in Chrome, Firefox, and Safari
5. Test at 375px (mobile) and 1440px (desktop)
6. Verify dark mode and light mode both work
7. Open a **Pull Request** using the PR template

## 📐 Code Style

- Use the existing CSS custom property tokens — don't hardcode colors
- Keep JavaScript vanilla (no libraries beyond Lucide)
- Add comments for any non-obvious logic
- Maintain semantic HTML5 structure
- Ensure ARIA attributes on interactive elements
